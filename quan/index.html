Programa

4 sesiones de 2 horas cada una, 1 por semana

Cada sesión combina demostración en vivo con casos reales de Quan y práctica guiada donde cada participante construye sus propios flujos.

S1

Claude Chat — Fundamentos

Semana 1👥 Todos📍 Básico
  • Arquitectura de prompts efectivos (rol, contexto, instrucción, formato)
  • Proyectos, memoria persistente y búsqueda web
  • Análisis de documentos: pólizas, contratos, términos
⚡ Optimización de licencias Claude
  • Selección estratégica de modelo según la tarea (Haiku, Sonnet, Opus): cuándo usar cada uno para maximizar velocidad, calidad y costo
  • Uso eficiente de contexto: técnicas para estructurar conversaciones que no desperdicien tokens ni mensajes
  • Gestión inteligente de archivos y documentos adjuntos para evitar reprocesamiento
  • Proyectos, instrucciones de sistema y artefactos como multiplicadores de productividad
Resultado

Proyecto de Claude configurado por área con instrucciones personalizadas, prompts reutilizables y criterio claro de cuándo usar cada modelo para maximizar la licencia.

S2

Claude para Office — Word, Excel, PowerPoint

Semana 2👥 Todos📍 Básico
  • Word: redacción de pólizas, comunicaciones, procedimientos
  • Excel: análisis de datos de ventas, recaudo y siniestralidad
  • PowerPoint: presentaciones para aliados y reportes ejecutivos
Resultado

Automatización de al menos 3 entregables recurrentes por persona con plantillas documentadas.

S3

Cowork, Tareas Programadas & Dispatch

Semana 3👥 Todos · énfasis Ops y Recaudo📍 Intermedio
  • Cowork: agente autónomo para archivos y carpetas locales
  • Tareas programadas y Dispatch: automatización con dependencias
  • Casos: reporte semanal de recaudo + monitoreo de vencimientos
Resultado

Cada participante configura al menos 1 tarea programada y 1 flujo de Cowork funcional.

S4

Claude Code + Claude en Chrome

Semana 4👥 Equipo técnico📍 Avanzado
  • Claude Code: instalación, configuración, desarrollo asistido, MCP Servers
  • Claude en Chrome: automatización tipo RPA de aplicaciones en navegador
  • Caso práctico: automatización RPA de un proceso real de Quan en Google Chrome
Resultado

Claude Code operativo + automatización RPA funcional en Chrome sobre un proceso real de Quan.
